Account summary viewer
------
Upload various monthly statements and generate a webpage to view certain statistics. You can currently upload both RBC and WealthSimple monthly statements. 
Then once the data is collected into one file this can be used to view balance history for specific accounts and date ranges.

To upload data from a Statements folder, run main.py

This project acts as a personal resource to manage finances for developers and couple Cassandra Litwinowich and Vixail Hadelyn.
